# Dot Metaverse - 파티클 원복 + 완전리셋 버튼

## 날짜
2026-07-10

## 내용
- 우주 파티클 확장 실험(스카이돔/박스균등/네모 파티클) 원복
- 원래 상태로 복구: 무지개 별 2500 + 큰 별 120 + 행성 4개 + 별똥별
- ⌖ 이동패드 가운데 = 완전리셋(이동+카메라+줌) 유지

## 복구 이유
- 별을 상하로 확장/개수 증가/스카이돔 시도했으나 네모 파티클이 지저분
- Three.js Points 기본 사각형 + 과도한 개수/크기로 "컬러 곰보" 발생
- 원래 무지개 별(구 분포)이 훨씬 깔끔하고 우주다움

## 현재 3D 지구본 최종 상태
- 무지개 셀 지구본 + 파티클(별 2500+120) + 행성4개 궤도 + 별똥별
- 정면 기준 좌표축 (열 위 A~S / 행 오른쪽 1~18), 클릭→그리드
- 회전패드(좌) + 이동패드(우, ⌖=완전리셋) + 줌슬라이더
- 셀/좌표/D3(3D교실) 팝업, 뒤로가기 팝업닫기
- 검색창 툴바, 2D 셀 전부 3D 반영

## 교훈 (향후 파티클 작업 시)
- 별은 원형 텍스처 + 적당한 개수(2500 수준)가 깔끔
- 과도한 확장/개수는 역효과
- 위아래 별 부족은 ⌖ 리셋으로 정면 유지하면 실사용 문제 없음
