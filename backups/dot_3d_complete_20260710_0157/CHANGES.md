# Dot Metaverse 3D 지구본 - 완성본 (시연용)

## 날짜
2026-07-10

## 전체 기능
### 3D 우주 지구본
- 324칸(18×18) 격자를 Three.js CSS3D 구면 매핑
- 실데이터 연동: presets + srvCells + cells (2D와 동일 소스, D3 등 반영)
- 빈 셀 위치기반 무지개 네온색, 소유 셀 아이콘+강조
- position:fixed 전체화면, window.innerWidth 리사이즈

### 우주 연출
- 무지개 별 파티클 2500개 + 큰 별 120개 (AdditiveBlending)
- 행성/위성 4개 (L2파랑/L3보라/L4청록/moon금색), 각기 궤도 공전+자전+발광
- 별똥별: 1.8초마다 확률적 유성, 무지개색 꼬리+페이드아웃

### 좌표축 (정면 셀 기준 동적)
- 열 A~S(금색) 화면 위, 행 1~18(청록) 지구본 오른쪽 곡면
- vector.project(camera)로 매 프레임 투영, 정면만 표시, 뒤쪽 숨김
- 좌표 클릭 → 열/행 셀 그리드 팝업

### 조작 컨트롤
- 왼쪽 하단: 회전 패드(▲◀■▶▼, 청색) - 지구본 제자리 회전
- 오른쪽 하단: 이동 패드(▲◀⌖▶▼, 주황) - 화면 위치 이동
- 상단 줌 슬라이더(#zsl 1~8) → 카메라 거리
- 마우스 드래그/휠도 유지

### 팝업
- 셀 클릭 → HTML 풀스크린 iframe (유튜브 영상 스타일)
- 좌표 클릭 → 열/행 셀 그리드 (유튜브 홈 스타일)
- D3 → 3D 바둑 교실(/dot/classroom3d.html) + 🚪입장 버튼
- S18 → 우주 space.html
- 뒤로가기 → 팝업만 닫힘 (history 연동)

### 검색
- 검색창을 툴바로 이동 (2D/3D 공통), L2/B3·이름·빈셀 검색

## 향후 과제
- 유저 셀 활성화 UX 간소화 (원클릭 템플릿, 인라인 편집) — 리텐션 핵심
- 이름·상호 등록만으로 그리드·지구본 자동 반영 (기반 완성됨)
- 3D 아바타 입장 연계 (openCellPopup 구조 활용)
- 지구본 초기 중앙 정렬 (현재 이동패드로 조정)
