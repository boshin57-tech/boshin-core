# 우주 바둑: 영역→행성 조립 (MVP 첫 조각)

## 날짜
2026-07-10

## 컨셉
"우주를 내 색으로 채우는 놀이" — 바둑이 휘발되지 않고 행성으로 남음
아이 대상: 규칙 학습보다 "내 행성이 커지는" 성취 감각

## 구현 (classroom3d.html)
- 🌍 행성 만들기 버튼 (컨트롤 바)
- buildPlanets(): stones로 흑/백 영역 계산 (floodFill 원리)
  - 빈 칸이 한 색에만 둘러싸이면 그 색 영역 + 돌 수
- 흑 영역 → 파란 바다행성(0x2266dd), 백 → 초록 대륙행성(0x22aa44)
- 행성 크기 = 영역, 발광 외곽, 회전 애니메이션
- AI 선생님 축하 (bubbleText + teacherSpeak)

## 향후 로드맵 (전문가 인사이트 기반)
1. [완료] 영역→행성 조립 (한 판의 마법)
2. 행성 조립 연출 강화 (돌들이 모여 행성 되는 애니메이션)
3. 내 은하계 저장 (dot.html에 행성 쌓기) — 수집·성장 루프
4. 행성 진화 (이기면 바다→대륙→도시→문명)
5. 별 수집 → 아바타·스킨 해금
6. 협동 모드 (친구와 우주 채우기, socket 기반)
7. 데일리 별자리 챌린지

## 핵심 설계 원리
- "결과물이 남는 바둑" — 매 판이 영구적 창조물 (마인크래프트/동물의숲 원리)
- 실력이 아니라 "내 우주가 자라는 것"이 리텐션 동력
- 패배 = "백이 더 큰 행성" 프레이밍 (AI 선생님)
